# Camp Connect

### 📢 캠핑장 찾기 어려우셨죠? 저희 캠프 커넥트가 연결 시켜드리겠습니다!

📆 개발 기간: 10/11 ~ 10/16

👨🏻‍🤝‍👨🏻 Front: 윤진선 & 이경아

👨🏻‍🤝‍👨🏻 Back: 김태웅 & 홍재환

🎯 개발 목표: 

1. 서로 다른 개발환경에서의 연동(CORS) 
2. 회원가입 & Spring에서 JWT 방식의 로그인 
3. 캠핑장 카테고리 필터링 
4. 캠핑장 리뷰 CRUD 기능
5. 캠핑장 예약 기능
6. 마이페이지 예약 현황 조회

✔ Tech Stack

| Stack     | Front        | Back                         |
| --------- | ------------ | ---------------------------- |
| Framework | React, Redux | Spring boot, Spring Security |
| Library   | Axios, Immer |                              |
| Database  |              | MySQL                        |
| Deploy    | S3           | AWS                          |
